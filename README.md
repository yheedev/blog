# https://yheedev.vercel.app

🌐 [English](README.en.md) | [日本語](README.ja.md)

Next.js와 TypeScript를 사용해서 개발한 개인 기술 블로그입니다.

개발했던 프로젝트, 디버깅 과정, 강의 및 책으로 공부한 내용, 코딩 테스트 풀이 등을 기록했습니다.

다국어 (한국어, 영어, 일본어)를 지원하고, SEO 최적화를 중심으로 개발했습니다. 마크다운 기반으로 블로그 포스트들을 작성했습니다.

### 실행 방법

```
yarn install
yarn dev
```
