# 레벨2: JadenCase 문자열 만들기

https://school.programmers.co.kr/learn/courses/30/lessons/12951

```js
function solution(s) {
  const arr = s.split(' ')

  for (let i = 0; i < arr.length; i++) {
    if (!arr[i]) continue

    const first = arr[i][0]
    const rest = arr[i].slice(1).toLowerCase()

    arr[i] = /[a-z]/i.test(first) ? first.toUpperCase() + rest : first + rest
  }
  return arr.join(' ')
}
```

## 공부/검색 내용

- 문자열 공백 기준으로 배열에 담으려면 `split(' ')` 사용
- 대문자인지 판별하기 위한 정규식 사용 (`/[A-Z]/.test(arr[i][0])`)
  - test는 정규식 객체의 메서드. 매치되면 true, 아니면 false 반환
- 숫자로 시작하는 단어의 나머지도 소문자로 바꿔줌

## 다른 코드

```js
function solution(s) {
  return s
    .split(' ')
    .map((v) => v.charAt(0).toUpperCase() + v.substring(1).toLowerCase())
    .join(' ')
}
```
